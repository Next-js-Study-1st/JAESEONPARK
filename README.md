## Commit Rule

| 커밋 타입     | 설명                           |
| ------------- | ------------------------------ |
| ✨ `feat`     | 새로운 기능 추가               |
| 🐛 `fix`      | 버그/오류 수정                 |
| 🔨 `chore`    | 코드/내부 파일/설정 수정       |
| 📄 `docs`     | 문서 수정 (README 등)          |
| 🪛 `refactor` | 코드 리팩토링 (기능 변경 없음) |
| 💻 `test`     | 테스트 코드 추가/수정          |
| 🎨 `style`    | 스타일 변경(포맷, 세미콜론 등) |
